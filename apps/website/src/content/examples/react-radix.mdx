A React 18 application using Radix UI primitives, bundled directly from npm for the browser. This shows how jiki can bundle and run real npm packages, including complex compound component libraries.

## Details

- Radix UI packages are bundled from npm with `bundlePackageForBrowser()`
- Includes Avatar, Dialog, and other Radix UI compound component patterns
- Radix UI's built-in accessible behavior and ARIA support work out of the box
- The React JSX runtime is shimmed for browser execution

## Try it yourself

1. Edit the component files to try different Radix UI primitives
2. Open dialogs and interact with the accessible components
3. Modify styles while keeping Radix behavior intact
