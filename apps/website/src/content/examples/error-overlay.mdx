A development error overlay that catches runtime exceptions and displays them with formatted stack traces. Covers error boundary patterns for unhandled errors, TypeErrors, and Promise rejections.

## How it works

- A global error handler catches uncaught exceptions
- Stack traces are formatted with source locations
- Error, TypeError, and unhandled Promise rejections are all handled
- A full-screen overlay shows error details with a dismiss action

## Things to try

1. Click the error-triggering buttons to see different error types
2. Examine the stack trace in the overlay
3. Dismiss the overlay and try a different error type
