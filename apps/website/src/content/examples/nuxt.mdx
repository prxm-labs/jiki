A Nuxt 3 application showcasing the framework's conventions: file-based routing, auto-imported composables, server API routes, layouts, and middleware. This is a full-stack Vue framework running in the browser.

## What this demonstrates

- The pages directory maps directly to URL routes
- Composables like `useCounter` and `usePosts` are auto-imported
- API endpoints are defined in the `server/api/` directory
- An auth guard middleware protects routes

## Try it yourself

1. Add a new file to the `pages/` directory to create a route
2. Create a composable in `composables/` and use it without importing
3. Define a server API route in `server/api/`
