- In-memory Vite configuration with the React plugin
- Full TypeScript support with type checking
- Fast Hot Module Replacement for rapid development
- React loaded from CDN with Vite's module resolution

1. Edit any component file and save to see instant HMR updates

3. Try modifying the Vite config for different build options
