A Jest-compatible test runner with a custom minimal test framework implementation. It runs multiple test suites using `describe`/`it`/`expect` syntax and parses the results into a visual display.

## What's inside

- A custom Jest-like framework providing `describe`, `it`, and `expect`
- Three test suites: calculator, user service, and event bus
- Stdout parsing that extracts pass/fail counts from the runner output
- Visual result display with pass/fail indicators per test

## Try it yourself

1. Click "Run Tests" to execute all test suites
2. Edit test files to add new test cases
3. Modify the source files and re-run to see failures
