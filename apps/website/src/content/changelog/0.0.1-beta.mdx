## Development releases

Summary of all beta milestones leading up to the 0.1.0 feature-complete beta.

### 0.0.1-beta.1 — Oct 19, 2025

Initial kernel with virtual filesystem and CommonJS module resolver. The in-memory VFS supports directory operations, symlinks, file watching, and snapshot serialization.

### 0.0.1-beta.2 — Nov 3, 2025

Shell implementation with command parsing, pipes (`|`), logical operators (`&&`, `||`), and built-in commands (`ls`, `cat`, `cd`, `pwd`, `mkdir`, `rm`, `echo`, `export`).

### 0.0.1-beta.3 — Nov 19, 2025

esbuild-wasm transpiler integration for TypeScript and JSX with lazy WASM loading and code transformation pipeline.

### 0.0.1-beta.4 — Dec 5, 2025

Container API and `boot()` entrypoint providing the main orchestration interface for initializing and managing the runtime.

### 0.0.1-beta.5 — Dec 28, 2025

Vite dev server with HMR and ESM-based module serving. Next.js dev server with file-based routing, Pages Router, and App Router support.

### 0.0.1-beta.6 — Jan 14, 2026

Hook-based plugin system with 6 hook types: `onResolve`, `onLoad`, `onTransform`, `onCommand`, `onInstall`, and `onBoot`. API mirrors esbuild conventions.

### 0.0.1-beta.7 — Jan 30, 2026

Resource limits and sandboxing for untrusted code, performance metrics tracking, network request interceptor for mocking `fetch()`, and heuristic-based TypeScript type checker.

### 0.0.1-beta.8 — Feb 12, 2026

Vite build configuration, package exports for CJS/ESM/types, and test infrastructure setup.

### 0.0.1-beta.9 — Feb 26, 2026

jiki-ui React component library: CodeEditor with Shiki syntax highlighting, Terminal with command history, BrowserWindow preview iframe, FileExplorer, AIChatPanel, and MobileTabBar.
