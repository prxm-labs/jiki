# Changelog

## v0.0.1 (2026-04-02)

First stable release with 30 examples, documentation website, and full integration test suite.

## v0.0.1-beta-5 (2026-03-13)

Plugin system, sandbox guard, performance metrics, network interceptor, TypeScript type checker, browser bundler, Container API, and React UI components (@run0/jiki-ui).

## v0.0.1-beta-4 (2026-02-14)

Shell with command piping and job control, dev server infrastructure, Vite/Next.js/SvelteKit/Remix framework integrations, service worker bridge, and error parsing.

## v0.0.1-beta-3 (2025-12-22)

Package management with npm registry client, semver dependency resolution, tarball extraction, pnpm content-addressable store layout, lockfile support, and workspace linking.

## v0.0.1-beta-2 (2025-11-28)

Module system with esbuild-wasm transpiler, ESM-to-CJS code transforms, Node.js-style module resolver, CommonJS kernel, and 24 Node.js polyfills.

## v0.0.1-beta-1 (2025-10-31)

Initial release with in-memory virtual filesystem, path/events/process/stream/assert polyfills, and utility functions.
